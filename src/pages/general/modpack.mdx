# Modpack

Xenyria offers an optional but recommended Fabric-based modpack.

The modpack includes both [XEEM](/general/xeem), our custom in-house client-side mod adding features specific to Xenyria, as well as various client-side performance and quality of life improvement mods.

## Installation

There are various ways to install the modpack. The recommended way is to use the [Prism Launcher](https://prismlauncher.org/).

### Prism Launcher

1. Download the [Prism Launcher](https://prismlauncher.org/).

2. Click "Add Instance" in the top left.

3. Click "Modrinth" on the left.

4. Search for "Xenyria Modpack" and select it.

5. Click "OK" in the bottom right.
