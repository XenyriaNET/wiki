# Preparation

## 1. Create a GitHub account

If you don't already have an account on GitHub, you'll need to create one. Go to [GitHub](https://github.com/) and click "Sign up" in the top right corner. Follow the instructions to create your account.

## 2. Navigate to the official repository of the Wiki.

Navigate to the official repository of the Wiki. You can find it [here](https://github.com/XenyriaNET/xenyria-wiki). This is the place that hosts both the content and the content of our Wiki.

## 3. Fork the Repository

Next up, you'll have to fork the repository. This gives you your own copy of the wiki where you're able to do edits. To do this, click the "Fork" button in the top right corner of the repository page. This will create a copy of the repository in your own account.

![Forking the repository](/assets/meta/guide1.webp)

## 4. Navigate to the directory of the page you want to edit

Navigate to the directory of the page you want to edit. You can do this by clicking the folders in the file list. Follow the following:

- Firstly, click the `src` folder.
- Secondly, click the `pages` folder.
- Thirdly, click the folder of the category the file is in (You can refer to the URL to find this out)
- Lastly, click the file you want to edit.
